# Agent Global Rules

이 프로젝트에서 에이전트가 반드시 따라야 하는 규칙.

## 커밋 전 필수 검증

코드를 커밋(`git commit`)하기 전에 **반드시** 아래 검증을 모두 통과해야 한다.
검증에 실패하면 커밋하지 않고 문제를 먼저 해결한다.

### 1. 빌드 검증
```powershell
yarn build
```
- 빌드 에러가 없어야 한다
- TypeScript 타입 에러가 없어야 한다

### 2. 린트 검증
```powershell
yarn lint
```
- ESLint 에러가 없어야 한다 (warning은 허용)

### 3. 브라우저 동작 확인 (UI 변경이 있는 경우)
- `yarn dev`로 개발 서버 실행
- 브라우저에서 변경된 화면이 정상적으로 렌더링되는지 확인
- 콘솔 에러가 없는지 확인

### 4. 테스트 실행 (테스트 파일이 존재하는 경우)
```powershell
yarn test
```
- 모든 테스트가 통과해야 한다

## 검증 실패 시 행동

1. 에러 내용을 분석하고 코드를 수정한다
2. 수정 후 검증을 다시 실행한다
3. 모든 검증이 통과할 때까지 반복한다
4. **검증이 모두 통과한 후에만 커밋한다**
